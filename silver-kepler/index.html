            📊 How We Calculate Scores

            🎯 Difficulty Level

            • Number of features/modules
            • Technology complexity (basic vs advanced)
            • Integration level (single vs multiple systems)
            • Learning required (known vs new tech)
            • Dependency level (APIs, hardware)

            📈 Success Probability

            • Scope vs available time
            • Team skill match
            • Data/resource availability
            • Demo feasibility
            • Risk count (integration, data risks)

            ✨ Uniqueness

            • Problem originality
            • Solution approach novelty
            • Technology combination rarity
            • Target user specificity
            • Existing solution similarity

            🧠 Tech Knowledge (Skills)

            • Required programming level
            • Domain knowledge depth
            • Algorithm/logic complexity
            • Tool/framework familiarity
            • System design understanding

            🛠️ Tech Stack Required

            • Frontend complexity
            • Backend complexity
            • Database/data handling level
            • AI/ML or advanced tech usage
            • Deployment & infrastructure needs
